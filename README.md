# StudyPal AI

AI-powered study assistant that helps students learn concepts, generate quizzes, summarize notes, and get AI-based explanations.

## Features
- AI Chat Assistant
- Notes Summarization
- Quiz Generation
- Study Recommendations
- Clean Streamlit UI

## Tech Stack
- Python
- Streamlit
- Google Gemini API
- Generative AI
- NLP

## Installation

git clone <repo-link>

pip install -r requirements.txt

streamlit run app.py
